## Program Project Overview

The **Fitness Center Performance Tracker System** is a comprehensive analytics project designed to help a gym track and analyze:

- Members
- Classes
- Attendance
- Trainers
- Membership types
- Payments and revenue

This project integrates database design, ETL processes, data modeling, reporting, and executive dashboards to turn raw fitness data into actionable insights for business decision-making.

---

## Project Scope & Objectives

The goal of this project was to design an end-to-end data analytics solution that demonstrates real-world skills used by data analysts.

**Key objectives included:**
- Designing a relational database structure
- Cleaning and transforming data through ETL
- Analyzing attendance and revenue trends
- Creating executive-level dashboards and reports
- Presenting insights clearly for stakeholders

---

## Tools & Technologies Used

- SQL / Relational Database Design
- Microsoft Access (database storage)
- Power BI Desktop & Power BI Service
- Power Query
- GitHub & GitHub Pages
- Data Modeling & DAX
- ETL Principles

---

## Key Analytics & KPIs

- Total active members
- Class attendance by class and difficulty level
- Monthly membership revenue
- Revenue by membership type
- Attendance patterns and participation trends
